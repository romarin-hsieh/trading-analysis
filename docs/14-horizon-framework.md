# 交易維度框架:三時間尺度 × 兩出手風格,六槽位實測 + 全面復盤 + 判斷依據

> /goal(2026-06-19):盤點交易維度 → 設計並實測「短(1-4週)/中(1-12月)/長(1-3年+)」×「低出手穩定 / 高出手 Sharpe>2」六槽位 → 年度閘門 = 該年 VOO 報酬 2 倍 → 復盤過去為何沒用 → 交付可重現的選股/進出場/停利停損判斷依據。產業鎖定:科技鏈/軟體/航太衛星/AI。

## 1. 交易維度盤點(taxonomy)

| 維度 | 可選值 | 我們實證過的結論 |
|---|---|---|
| **總經 regime** | 升息/降息、QE/QT、通膨、成長/衰退 | regime 標籤有描述力,**當開關會扣分**(docs/09 §七);唯一可靠的 regime 訊號=品質因子在壓力期最強 |
| **市場局面** | 多頭/空頭/盤整、高低波動、廣度 | 2015-24 樣本只有 ~3-4 個真空頭;局面辨識滯後(200SMA 慢 10-25 天,Kalman 快但 whipsaw) |
| **領域/產業** | 科技鏈/軟體/航太衛星/AI/防禦性 | 科技 beta 本身就是十年最大報酬來源;產業內選股 alpha 幾乎不存在(§3 分解) |
| **時間尺度** | 短 1-4 週 / 中 1-12 月 / 長 1-3 年+ | **日線+免費資料只支撐中長線**;短線被成本牆與資料頻率鎖死(§2 S 槽) |
| **交易風格** | 趨勢/動量、均值回歸、事件、套利、配置 | 動量=beta+(有衰退);均值回歸(pairs)套利殆盡;事件(PEAD/insider)無漂移;**配置/風險塑形=唯一顯著 alpha** |
| **出手頻率** | 年 1 次 ~ 日 250 次 | 頻率越高、成本牆越致命(overnight gross Sharpe 0.89 → net **−0.97**) |
| **交易目標** | 絕對報酬 / 相對大盤 / 回撤控制 | 50-100% CAGR 低風險=數學不可達(Calmar 牆);**回撤控制是可交付的那半** |

## 3. 決定性分解:過閘門的是「策略」還是「beta+清單」?

對照組(零策略):**QQQ 純買進持有 = 4/10 過閘門**(2018/2020/2022/2023),CAGR +19.5%,Sharpe 0.91。

```
過閘門成分分解:
  科技產業 beta(QQQ B&H)          = 4/10   ← 買個 ETF 就有
  + 47 檔清單的 curation(近因偏誤) = +3     ← L-lo 年度等權 7/10(1 次決策/年!)
  + 動量規則的增量                  = ~0     ← M-lo/M-hi 6-7/10 ≈ 同清單零策略等權
```

**鐵證:一年只做一次決策的「零策略等權」(L-lo)追平所有精緻策略槽。** 過閘門的功勞屬於「持有科技」和「清單是 2026 視角挑的」,不屬於任何規則。2024 M-lo 的 +95.8% 就是 curation 的極端例子(top-5 集中壓中 NVDA/SMCI 世代)。

## 4. 兩道閘門的數學現實(誠實判決)

- **「每年 > 2×VOO」**:沒有任何槽位 10/10。最佳 7/10,且增量=beta+curation。這個閘門在 +25% 的大盤年要求 +50%,等效要求常態 Calmar>2 —— 我們已證明可投資前沿 ~0.7(docs/07)。**在此資料上,「每年 2×VOO」不是策略問題,是數學不可達**;能週期性通過的只有「集中科技 beta 在其大年」。
- **「高出手且 Sharpe>2」**:十年期沒有任何槽位 >1.21。單一年份可達(combo 2025 = 2.41),但單年=單一路徑。**高出手方向,成本牆先於 Sharpe 出現**(S-hi:毛 0.89 → 淨 −0.97)。Sharpe>2 持續十年 = 頂級對沖基金水準,需要日內/另類資料維度,非日線免費資料可及。

## 5. 全面復盤:過去的方式為什麼「沒用」?

**Q:是研究方式錯、驗證方式錯,還是其他?**

### 5a. 驗證方式錯了嗎?——**沒有。驗證是整個專案唯一一直在「贏」的部分。**
驗證抓到 30+ 個真錯誤:sizing 排序相依(12.4pp artifact)、CV embargo 洩漏、regime 歸因假陽性(FDR)、regime-rotation in-sample 雜訊(13 agent 證偽)、倖存者 +126bps、t-stat 自相關灌水 3.7×、long-only sleeve 的 beta 假象(零訊號控制)、選股清單 curation(+62.8% 幻覺)。**每一次拿掉驗證,數字都「變好看」——那正是幻覺的定義。**驗證沒有殺死可行策略;它殺死的全是假象。

### 5b. 研究方式錯了嗎?——**早期有真 bug,已修;修正讓結果「變誠實」而非「變差」。**
等股數 sizing、未用除息還原價、order-dependence、SPA dead code——這些是實作錯誤,被三輪 error-hunt 抓出。修正後的方法鏈(因子閘門 → 零訊號控制 → 消融實驗 → 對抗驗證)是**正確且可重現的研究方法**,本檔的六槽位就是用它跑的。

### 5c. 真正的原因(其他)——四個結構性事實:
1. **目標超出資料的數學上限**:Calmar 牆(可達 0.7 vs 需 3.3)、Sharpe 天花板(~1.2 vs 需 >2)、2×VOO 每年(最佳 7/10 且是 beta)。三個目標三面牆,同一個根源:**日線+免費資料的資訊含量有限(Grinold: IR=IC×√breadth)**。
2. **Alpha 衰退**:Minervini Sharpe 十年砍半(1.17→0.64)、insider 死、PEAD 死、TOM 死——公開的 edge 被套利掉(McLean-Pontiff:發表後 −58%)。你自己在最初 /goal 的預測完全正確。
3. **資料維度是綁定約束**:三根免費槓桿(延長歷史/新 IC 源/修偏誤)全拉過,天花板不動。要突破需要日內(ORB)、選擇權(VRP)、分析師修正——都在預算外。
4. **偏誤系統性製造「看起來可行」**:倖存者、curation、in-sample、成本忽略——每個都讓壞策略看起來像好策略。這不是我們的方式「沒用」,而是**大多數網路上「有用」的展示,靠的正是這些偏誤**。

> **一句話:方式沒有錯——是方式「正確到足以證明目標在此資料上不可達」。這是研究的成功,不是失敗。**

## 6. 可重現的判斷依據(選股/進場/出場/停利/停損 rulebook)

以下全部來自通過驗證的元件(標注可靠度)。適用:科技/軟體/航太衛星/AI,月度操作節奏。

### 選股(篩選器,非預測器)
1. **宇宙機械化**:指數成分或「流動性>$5 + 市值門檻」機械篩選。**絕不回測手挑清單**(§3 的教訓)。
2. **結構過濾**(可靠):Trend Template——價 > 50SMA > 150SMA 堆疊 + 距 52 週高 <25%。
3. **相對強弱**(可靠):RS 百分位 ≥70(對同宇宙)。
4. **品質**(唯一有預測力的基本面):gross_profitability(毛利/資產)宇宙前 40%;**避開**極高營益率擁擠股(反號證據)。
5. **競品比較**:同 peer 群內比 RS / GP / 盈餘殖利率百分位(`factors/validation` 現成)。

### 進場
- **月度決策點**進場(月初 rebalance 日),不做日內擇時。
- 不追高:進場時價距 50SMA < 1.5×ATR(126d),否則等回測。
- 高波動股(年化 vol>40%):**分散進**(K≥20 檔的籃子),單檔 ≤5%。

### 出場(最重要的一節——負面清單是驗證最強的部分)
- **月度 rank-out**:跌出動量前 2K 名才出(hysteresis,§11 V8),避免月中情緒單。
- 高波動股結構切:收盤 < 126 日低點 → 下月初出(death-spiral cut)。
- **❌ 禁用:日頻 50SMA 停損**(高波動股 Sharpe 0.48→0.9 的元兇)、**❌ 禁用:SPY 200SMA 全現金擇時**(錯過 V 反彈的元兇)。

### 停利
- 動量部位**不設硬停利**(讓贏家跑,rank-out 自然收割)。
- 短 swing(若做):triple-barrier 對稱柵欄(±2×ATR,21d 時間柵欄)——框架可靠,期望值不保證。

### 停損(兩層)
- **個股層**:上面的月度結構切(126d low),不用日頻。
- **組合層**(真正的保命機制):**回撤預算 L 刻度**——先選你能忍的 MDD,讀出配置:L=0.4→MDD −7%、L=1.0→−19%、L=2.0→−38%(`scripts/defensive_overlay.py` 印當日權重)。

### 部位
- 等權或逆波動(兩者差異二階);單股上限 10%;帳戶層 vol>40% 的股票總曝險 ≤50%。
