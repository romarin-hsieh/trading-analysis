# TR-09 Black-Scholes 選擇權定價 — N/A(不可測,文件化)

## 1. 機制定義與理論
Black-Scholes-Merton(1973):在「標的服從幾何布朗運動(GBM)、波動率恆定、無摩擦連續避險」假設下,歐式選擇權有封閉解定價 C = S·N(d1) − K·e^(−rT)·N(d2)。理論貢獻是**無套利複製**:選擇權價值來自動態對沖組合的成本,不是預期報酬。出處:Black & Scholes (1973), Merton (1973);諾貝爾獎 1997。

## 2. 相關既有機制
- TR-05(GBM Monte Carlo)直接檢驗 BSM 的**分布假設**在科技股上的失真程度。
- docs/13 §1 目錄的 VRP/Wheel/covered-call 策略(需選擇權資料,同樣未測)。
- Serenity 的選擇權賣方 swing(docs/16)——不可觀測,同一資料缺口。
