- **同窗縮水只有 +2%**(q4 甚至 −10%,比 Carhart 還高)。初版報的「縮水 30%」是**混窗 CONFIRMED-BUG**:
  分母 5.90% 是 131 個月的數字,q 檔止於 2024-12,被截掉的 17 個月(2025-01..2026-05)年化 +22.7%
  的熱尾撐高了分母。稽核抓出後,腳本已加同窗 Carhart 列。
- **t 1.77<2.0 是截斷窗口的統計 power 問題**(同窗 Carhart 自己 OLS t 也只有 1.87),不是 q 吸收;
  HAC 下 q5 的 2.21 甚至高於同窗 Carhart 的 2.17。
- **IA beta −0.32(t −3.8)真且可解釋**:子期穩定(−0.55/−0.26);sleeve 歸因=TQQQ(−1.15)與防禦輪動

- F0 字面註記:預先承諾的 t≥2.0 沒指明 OLS/HAC——依 house 慣例(docs/03 §O7 HAC)判 ROBUST;
  依 OLS 字面則是 underpowered。正式標籤:**ROBUST-in-magnitude / underpowered-window**。
